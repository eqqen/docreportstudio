Политика конфиденциальности

Действует с 23 мая 2026 года

Главное в одном абзаце. Сервис не собирает данные пациентов. Заключения формируются в браузере на устройстве пользователя и в базу данных Сервиса не передаются. В базе хранится только адрес электронной почты пользователя, история открытия инструментов и тексты обратной связи, отправленные пользователем явно.

1. Какие данные собираются

В рамках работы Сервиса собираются и хранятся следующие данные:

  • Адрес электронной почты — для аутентификации (вход по magic-link) и связи с пользователем.
  • События использования — факт открытия дашборда и отдельных инструментов (без содержимого работы). Используется для понимания общей активности.
  • Обратная связь — текст и категория сообщения, которые пользователь явно отправляет через форму «Обратная связь».
  • User-Agent браузера — для диагностики совместимости при отправке обратной связи.

2. Какие данные НЕ собираются

Сервис не собирает и не хранит:

  • персональные данные пациентов пользователя;
  • описания находок и тексты сформированных пользователем заключений;
  • изображения МРТ-исследований;
  • пароли (Сервис не использует пароли — вход по одноразовой ссылке на e-mail).

3. Аналитика лендинга

На главной странице (docreportstudio.com) установлен счётчик «Яндекс.Метрика» для оценки источников трафика. Сбор Вебвизора и анализа форм отключён; cтрого внутри приложений ШОП и ПОП (страницы /app/cervical и /app/lspine) аналитика не работает — это сознательное решение.

4. Где хранятся данные

База данных размещена в облаке Supabase в регионе eu-central-1 (Франкфурт, Германия). Доставка электронных писем осуществляется через сервис Resend. Хостинг сайта — Cloudflare Pages (глобальная CDN). Все указанные провайдеры соответствуют требованиям GDPR.

5. Кто имеет доступ

Доступ к собранным данным имеет только владелец Сервиса. Данные не передаются третьим лицам, не продаются и не используются для рекламы. Сервис не размещает рекламу.

6. Сроки хранения

Данные хранятся до тех пор, пока существует учётная запись пользователя. По запросу пользователя учётная запись и связанные с ней данные удаляются в течение 14 рабочих дней.

7. Права пользователя

Пользователь имеет право:

  • запросить выгрузку всех своих данных;
  • запросить удаление учётной записи и всех связанных данных;
  • отозвать согласие на обработку данных в любое время.

Для реализации этих прав напишите на info@docreportstudio.com.

8. Файлы cookie

Сервис использует технические cookie, необходимые для поддержания авторизованной сессии (Supabase Auth). Маркетинговые и трекинговые cookie не используются.

9. Контакты

Вопросы по обработке персональных данных: info@docreportstudio.com
